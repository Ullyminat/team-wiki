# Team Knowledge Base (Team Wiki)

**Team Wiki** — это современное веб-приложение для хранения, организации и совместного использования знаний внутри команды.

---

## 🚀 Быстрый старт через Docker (Рекомендуется)

Для запуска проекта вам понадобится только установленный [Docker](https://www.docker.com/).

### Запуск одной командой
В корне проекта выполните:
```bash
docker compose up --build -d
```

После завершения сборки:
1.  **Приложение**: [http://localhost:3000](http://localhost:3000)
2.  **База данных (Adminer)**: [http://localhost:8081](http://localhost:8081)
3.  **Хранилище файлов (MinIO)**: [http://localhost:9001](http://localhost:9001)

### Данные для входа в инфраструктуру
- **PostgreSQL**: Пользователь: `wiki_user` | Пароль: `wiki_password` | База: `team_wiki` | Сервер: `db`
- **MinIO**: Пользователь: `admin` | Пароль: `password123`

---

## ✨ Возможности
* 📂 **Иерархия страниц**: Бесконечная вложенность документов.
* 📝 **Markdown**: Полноценный редактор с предпросмотром.
* 👥 **Командная работа**: Создание воркспейсов, приглашение коллег.
* 🕰️ **Версии**: История изменений каждой страницы и откат к любой версии.
* 🚀 **Поиск**: Мгновенный поиск по контенту и тегам.
* 📎 **Облачные вложения**: Загрузка файлов через MinIO/S3.
* 🌍 **Explore**: Обзор публичных пространств организации.
* 🔒 **Приватность**: Настройка доступа к личным и командным пространствам.

## 🛠 Технологии
- **Next.js 15+** (App Router)
- **Prisma ORM** (PostgreSQL)
- **NextAuth.js v5** (Безопасность)
- **MinIO** (Облачное хранилище)
- **Tailwind CSS** (Стиль)
- **Docker** (Контейнеризация)

---

## 👨‍💻 Разработка (Локально)

Если вы хотите вносить правки в код в режиме реального времени:

1.  Поднимите инфраструктуру: `docker compose up -d db minio`
2.  Создайте `.env.local` на основе `.env`.
3.  Установите зависимости: `npm install`
4.  Запустите: `npm run dev`
